This package is about spv.
==========================

PyPi: https://pypi.org/manage/projects/

สวัสดีจ้าาาา แพ็คนี้คือแพ็คเกจที่อธิบาย spv.

วิธีติดตั้ง
~~~~~~~~~~~

เปิด CMD / Terminal

.. code:: python

   pip install mynameisspv

วิธีใช้งานแพ็คเพจนี้
~~~~~~~~~~~~~~~~~~~~

-  เปิด IDLE ขึ้นมาแล้วพิมพ์…

.. code:: python

   from mynameisspv import Spv

   ss = Spv()
   ss.show_name()
   ss.show_page()
   ss.about()
   ss.picsart()
